## ☁️ Microsoft Sentinel Threat Hunting & Command-and-Control Detection Lab

This project demonstrates the deployment of a hybrid security monitoring environment using Azure Arc, Microsoft Sentinel, Azure Monitor Agent (AMA), KQL, and Azure Machine Learning.

### Skills Demonstrated

- Microsoft Sentinel
- Azure Arc
- Threat Hunting
- KQL
- Incident Investigation
- MITRE ATT&CK
- Azure Machine Learning
- MSTICPy

### Key Activities

- Connected an on-premises server to Azure using Azure Arc
- Ingested security telemetry into Microsoft Sentinel
- Simulated DNS-based Command-and-Control activity
- Created hunting queries using KQL
- Developed Near Real-Time analytics rules
- Conducted investigations using Sentinel Investigation Graphs
- Integrated Azure Machine Learning and MSTICPy
